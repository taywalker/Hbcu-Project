Algorithms

Algorithms – Step-by-step solution using a set of specified rules to solve a problem. We use algorithms daily such as making a sandwich. However, there can be more than one way to make a sandwich. It’s the same with writing algorithms to solve a problem. Every programmer will solve a problem differently than the next programmer, but all have the same goal in common, to solve the problem!
