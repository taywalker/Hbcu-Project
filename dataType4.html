Syntax

Syntax - Refers to the set of rules that defines the placement and order of data types when writing code

We think of Syntax as a set of rules that we have been given that must be followed. Python returns a syntax error such as the one in the picture when the programmer has violated these set rules. The syntax error will show exactly where and why the error is occuring. Violation of these rules will result in the code not being able to execute the directions the programmer has given it.
